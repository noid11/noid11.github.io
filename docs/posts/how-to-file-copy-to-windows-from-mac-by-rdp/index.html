RDP を使って Mac から Windows にファイルをコピーする方法

  • クリップボードにコピーして貼り付るのが手軽で良い
    1. Mac 側でコピーしたいファイルを選択し、クリップボードにコピー
    2. RDP で Windows に接続し、ファイルを送信したいディレクトリをよしなに選択してペースト
  • RDP 設定の Devices & Audio で Clipboard にチェックが入っていることを確認すること
